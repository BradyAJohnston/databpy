# LinkedObjectError

``` python
LinkedObjectError(message)
```

Error raised when a Python object doesn’t have a linked object in the 3D scene.

## Parameters

| Name | Type | Description | Default |
|----|----|----|----|
| message | [str](https://docs.python.org/3/library/stdtypes.html#str) | The error message describing why the linked object is missing or invalid. | *required* |

## Attributes

| Name | Type | Description |
|----|----|----|
| message | [str](https://docs.python.org/3/library/stdtypes.html#str) | The error message that was passed. |
